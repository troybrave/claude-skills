# Lyric Formatting Guide

Complete reference for formatting lyrics for Suno AI.

---

## Section Markers

### Standard Sections

```
[Intro]
[Verse]
[Verse 1]
[Verse 2]
[Pre-Chorus]
[Chorus]
[Post-Chorus]
[Bridge]
[Outro]
```

### Instrumental Sections

```
[Instrumental]
[Instrumental Break]
[Interlude]
[Guitar Solo]
[Synth Solo]
[Piano Solo]
[Drum Break]
[Breakdown]
[Build]
[Drop]
```

### Specialized Sections

```
[Hook]
[Refrain]
[Tag]
[Ad-lib]
[Spoken]
[Rap Verse]
```

## Proper Spacing

### Between Sections (REQUIRED)

```
[Verse 1]
Walking through the city lights
Shadows dancing on the walls
Every step feels like a fight
But I won't let myself fall

[Pre-Chorus]
Building up inside of me
A fire I can't contain
```

### Within Sections (NO extra lines)

```
[Chorus]
Rise up, rise up
We're breaking through the night
Rise up, rise up
Into the morning light
```

### Bad Spacing (AVOID)

```
[Chorus]

Rise up, rise up

We're breaking through the night

Rise up, rise up
```

## Common Mistakes

### Empty Sections
```
[Intro]

[Verse 1]
```
Fix: Add content or remove marker.

### Missing Blank Lines
```
[Verse 1]
Line one
[Chorus]
Chorus line
```
Fix: Add blank line between sections.

### Inconsistent Capitalization
```
[verse 1]
[CHORUS]
[Pre-chorus]
```
Fix: Use Title Case consistently: `[Verse 1]`, `[Chorus]`, `[Pre-Chorus]`

### Overloaded Vocal Tags
```
[whispered] [soft] [intimate] Words here
```
Fix: Use one primary tag: `[whispered] Words here`

### Contradictory Tags
```
[whispered] [belted] Sing it loud
```
Fix: Choose one delivery style.

## Pro Tips

### Syllable Density
- **Fast sections**: Short words, simple rhymes
- **Slow sections**: Can handle longer phrases
- **Rap verses**: Higher syllable count per line

### Natural Phrasing
Write lyrics as you'd naturally sing them:
```
Good: Running through the rain tonight
Bad: I am running through the rain on this night
```

### Breath Markers
For precise phrasing control:
```
Running through the rain [breath] finding my way home
```

### Hook Repetition
Choruses should repeat key phrases:
```
[Chorus]
Light it up, light it up
Set the whole world on fire
Light it up, light it up
Taking us higher and higher
```
