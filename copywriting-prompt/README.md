# Copywriting Prompt

Creates precisely optimized prompts that generate high-converting marketing copy for Claude or ChatGPT.

## What It Does

Give it your product, audience, and goal — it builds a framework-backed prompt that generates sales pages, emails, ads, and social posts.

## Frameworks Included

- **AIDA** — Attention, Interest, Desire, Action
- **PAS** — Problem, Agitate, Solution
- **PASTOR** — Problem, Amplify, Story, Transformation, Offer, Response
- **4Ps** — Promise, Picture, Proof, Push

## Supported Copy Types

| Type | Best Framework | Key Element |
|------|---------------|-------------|
| Landing Page | AIDA or PAS | Strong headline |
| Email Sequence | Welcome / Launch / Nurture | Subject line |
| Facebook/Google Ad | Hook-Problem-Solution | First line hook |
| Social Media Post | Hook-Value-CTA | Opening hook |
| Sales Letter | PASTOR | Story + proof |

## Installation

1. Copy the `copywriting-prompt/` folder into `~/.claude/skills/`
2. The skill is automatically available in Claude Code

## Usage

```
/copywriting-prompt
```

Or ask Claude: "Create a copywriting prompt for my landing page"

## Requirements

- [Claude Code](https://claude.ai/claude-code)
- No additional dependencies
